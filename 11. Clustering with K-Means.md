# 11. Clustering with K-Means

> Apply the K-Means algorithm to cluster a dataset (e.g., customer segmentation) and visualize the clusters.

---

## 1. K-Means in WEKA

WEKA provides **SimpleKMeans** under the Cluster tab.

### Steps:

1. Open WEKA → Explorer → **Preprocess**.
2. Load a dataset (e.g., `iris.arff` or customer purchase dataset).
3. Go to the **Cluster** tab.
4. Choose **SimpleKMeans** (`weka.clusterers.SimpleKMeans`).
5. Set number of clusters (e.g., `k = 3`).
6. You can choose distance function (default: Euclidean).
7. Run clustering → WEKA will output cluster centroids and assign each instance to a cluster.
8. Click **Visualize** → Scatter plot of clusters.

### Example (Iris dataset):

KMeans finds 3 clusters corresponding roughly to the 3 species of iris flowers.

---

## 2. K-Means in Python (scikit-learn)

### Load Dataset

```python

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# Load iris dataset
iris = load_iris()
X = iris.data
y = iris.target
```

### Preprocess Data (Optional: Normalize)

```python
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
```

### Apply K-Means Clustering

```python
# Set number of clusters
kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
kmeans.fit(X_scaled)

# Get cluster labels and centroids
labels = kmeans.labels_
centroids = kmeans.cluster_centers_

print("Cluster Labels:", labels)
print("Centroids:\n", centroids)
```

### Visualize Clusters

```python
# Plot first 2 features
plt.figure(figsize=(10, 6))
plt.scatter(X_scaled[:, 0], X_scaled[:, 1], c=labels, cmap='viridis', marker='o', alpha=0.6)
plt.scatter(centroids[:, 0], centroids[:, 1], c='red', marker='X', s=200, label='Centroids')
plt.xlabel('Sepal Length (scaled)')
plt.ylabel('Sepal Width (scaled)')
plt.title('K-Means Clustering on Iris Dataset')
plt.legend()
plt.show()
```

### Determine Optimal K using Elbow Method

```python
inertia = []
K_range = range(1, 10)

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X_scaled)
    inertia.append(kmeans.inertia_)

# Plot elbow curve
plt.figure(figsize=(8, 5))
plt.plot(K_range, inertia, marker='o')
plt.xlabel('Number of Clusters (k)')
plt.ylabel('Inertia (Within-cluster sum of squares)')
plt.title('Elbow Method for Optimal k')
plt.grid()
plt.show()
```

## 3. Customer Segmentation Example

### Sample Customer Dataset

```python
# Create synthetic customer data
data = {
    'Annual_Income': [15, 16, 17, 18, 19, 50, 51, 52, 53, 54, 90, 91, 92, 93, 94],
    'Spending_Score': [39, 81, 6, 77, 40, 76, 6, 94, 3, 72, 14, 99, 15, 42, 87]
}

df = pd.DataFrame(data)
X = df.values
```

### Apply K-Means

```python
kmeans = KMeans(n_clusters=3, random_state=42)
df['Cluster'] = kmeans.fit_predict(X)
```

### Visualize Customer Segments

```python
plt.figure(figsize=(10, 6))
plt.scatter(df['Annual_Income'], df['Spending_Score'], c=df['Cluster'], cmap='rainbow', marker='o', s=100)
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], c='black', marker='X', s=300, label='Centroids')
plt.xlabel('Annual Income (k$)')
plt.ylabel('Spending Score (1-100)')
plt.title('Customer Segmentation using K-Means')
plt.legend()
plt.show()
```
